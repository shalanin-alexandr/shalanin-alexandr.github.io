# KROK - Landing Page

Минималистичный темный landing page для вашего приложения.

## Структура проекта

```
KROK website/
├── index.html      # Основная HTML-структура
├── styles.css      # Стили с темным дизайном
├── script.js       # JavaScript для интерактивности
└── README.md       # Этот файл
```

## Секции сайта

1. **Hero** - Главный экран с заголовком и кнопками
2. **Возможности** - Сетка из 4 карточек с функциями приложения
3. **Скриншоты** - Галерея скриншотов (плейсхолдеры для замены)
4. **Преимущества** - Список из 3 преимуществ с нумерацией
5. **Команда** - Карточки членов команды
6. **FAQ** - Раскрывающиеся вопросы-ответы
7. **Контакты** - Контактная информация и социальные сети

## Как заполнить контентом

### Текст
Откройте `index.html` и замените:
- Заголовки и описания в каждой секции
- Названия функций и их описания
- Вопросы и ответы в FAQ
- Контактную информацию

### Скриншоты
Замените плейсхолдеры в секции `#screenshots`:
```html
<div class="screenshot-placeholder">
  <!-- Замените на: -->
  <img src="path/to/screenshot.png" alt="Описание">
</div>
```

### Фотографии команды
Замените SVG иконки в секции `#team`:
```html
<div class="team-avatar">
  <!-- Замените на: -->
  <img src="path/to/photo.jpg" alt="Имя">
</div>
```

### Логотип
Замените текст "KROK" в навигации на:
```html
<a href="#" class="logo">
  <img src="path/to/logo.png" alt="KROK">
</a>
```

### Ссылки
Обновите ссылки:
- Кнопки "Скачать" и "Узнать больше"
- Социальные сети в футере
- Email и телефон в контактах

## Цветовая схема

Вы можете изменить цвета в `styles.css`:

```css
:root {
    --bg-primary: #0a0a0a;      /* Основной фон */
    --bg-secondary: #141414;    /* Вторичный фон */
    --bg-tertiary: #1a1a1a;     /* Фон карточек */
    --text-primary: #ffffff;    /* Основной текст */
    --text-secondary: #a0a0a0;  /* Вторичный текст */
    --accent: #6366f1;          /* Акцентный цвет */
    --accent-hover: #4f46e5;    /* Акцент при наведении */
    --border: #2a2a2a;          /* Цвет границ */
}
```

- Hover-эффекты

## Запуск

Просто откройте `index.html` в браузере или используйте локальный сервер:

```bash
# Python 3
python -m http.server 8000

# Или используйте Live Server в VS Code
```

Затем откройте http://localhost:8000
