Введение: Что такое CSS?

Если HTML — это скелет вашего сайта, то CSS — это его кожа, одежда и макияж. CSS (Cascading Style Sheets) превращает скучные серые страницы в красивые, современные интерфейсы.

Без CSS

Простой белый текст на темном фоне

С CSS

Стилизованный текст с красивым фоном

Ваша цель на этот урок: научиться подключать CSS к HTML и освоить базовые стили: цвета, шрифты, отступы и выравнивание.

1. Подключение CSS: 3 способа

Есть три основных способа добавить стили к вашей HTML-странице:

Способ 1: Встроенные стили (Inline)

Стили пишутся прямо в атрибуте style HTML-тега.

<h1 style="color: blue; font-size: 24px;">Заголовок</h1>

Плюсы: быстро, просто

Минусы: сложно поддерживать, нельзя переиспользовать

Способ 2: Внутренние стили (Internal)

Стили пишутся в теге <style> внутри <head>.

<head>
    <style>
        h1 {
            color: blue;
            font-size: 24px;
        }
    </style>
</head>

Плюсы: все стили в одном месте

Минусы: работает только для одной страницы

Способ 3: Внешние стили (External)

Стили пишутся в отдельном файле с расширением .css и подключаются через <link>.

Структура файлов:

my-site/
├── index.html
├── about.html
└── styles.css 
<!-- В HTML-файле -->
<head>
    <link rel="stylesheet" href="styles.css">
</head>

/* В styles.css */
h1 {
    color: blue;
    font-size: 24px;
}

Плюсы: переиспользование, легкая поддержка, кэширование

Минусы: дополнительный файл

Рекомендация: всегда используйте внешние стили для реальных проектов!

2. Селекторы: Как обратиться к элементам

Селекторы — это способ сказать браузеру, к каким элементам применить стили.

Селектор по тегу

p { color: red; }

Применяется ко всем тегам <p>

Селектор по классу

.highlight { background: yellow; }

Применяется к элементам с class="highlight"

Селектор по ID

#header { height: 100px; }

Применяется к элементу с id="header"

/* Селектор по тегу */
h1 {
    color: #2c3e50;
}

/* Селектор по классу */
.text-red {
    color: red;
}

/* Селектор по ID */
#main-header {
    font-size: 2.5rem;
}

/* Псевдокласс :hover */
button:hover {
    background-color: #2980b9;
}

Псевдокласс :hover - реакция на наведение

Псевдокласс :hover позволяет применять стили, когда пользователь наводит курсор на элемент. Это основа для создания интерактивных интерфейсов! 

/* Простой hover эффект */
button:hover {
    background-color: #2980b9;
    transform: scale(1.05);
}

/* Hover для ссылок */
a:hover {
    text-decoration: underline;
    color: #e74c3c;
}

/* Hover с плавной анимацией */
.card {
    transition: all 0.3s ease;
}

.card:hover {
    transform: translateY(-5px);
    box-shadow: 0 10px 20px rgba(0,0,0,0.2);
}

3. Цвета и фон: Первое преображение

Цвета — самый простой способ изменить внешний вид сайта. В CSS есть несколько способов задать цвет:

red
#3498db
rgb(46,204,113)
rgba(155,89,182,0.7)
/* Разные способы задания цвета */
.element {
    color: red; /* Название цвета */
    color: #ff0000; /* HEX */
    color: rgb(255, 0, 0); /* RGB */
    color: rgba(255, 0, 0, 0.5); /* RGB с прозрачностью, где последнее значение 0.5 - прозрачность */
    
    background-color: #f8f9fa;
    background-image: url('image.jpg');
}

4. Текст и шрифты: Читаемость и красота

Шрифты сильно влияют на восприятие вашего сайта. CSS предоставляет множество свойств для работы с текстом. 

p {
    /* Шрифт */
    font-family: 'Arial', sans-serif;
    font-size: 16px;
    font-weight: bold; /* normal, bold, 100-900 */
    
    /* Выравнивание и интервалы */
    text-align: center; /* left, right, center, justify */
    line-height: 1.6;
    letter-spacing: 1px;
    
    /* Декорации */
    text-decoration: underline; /* none, underline, line-through */
    text-transform: uppercase; /* lowercase, capitalize */
}

Пример стилизованного текста

Этот текст демонстрирует различные CSS-свойства для работы со шрифтами. Обратите внимание на выравнивание, межстрочный интервал и гарнитуру шрифта.

5. Блочная модель: Margin, Padding, Border

Каждый элемент в CSS представлен как прямоугольный блок. Понимание блочной модели — ключ к верстке.

Строение блочной модели

Content — Содержимое элемента (текст, картинка)
Padding — Внутренние отступы (пространство между контентом и границей)
Border — Граница элемента
Margin — Внешние отступы (пространство между элементами) 
.box {
    /* Размеры */
    width: 300px;
    height: 200px;
    
    /* Внутренние отступы */
    padding: 20px; /* Со всех сторон */
    padding: 10px 20px; /* Вертикально Горизонтально */
    padding: 10px 20px 15px 5px; /* Верх Право Низ Лево */
    
    /* Граница */
    border: 2px solid #3498db;
    border-radius: 8px; /* Закругленные углы */
    
    /* Внешние отступы */
    margin: 20px auto; /* Центрирование блока */
}

Демонстрация блочной модели

Этот блок имеет внутренние отступы (padding), границу (border) и внешние отступы (margin).

6. Практика: Стилизуем страницу "О себе"

Давайте применим все изученное на практике и стилизуем HTML-страницу, созданную в прошлом уроке. 

/* styles.css */
/* Сброс стандартных отступов */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    font-family: 'Arial', sans-serif;
    line-height: 1.6;
    color: #333;
    background-color: #f4f4f4;
    padding: 20px;
}

header {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    color: white;
    padding: 40px 0;
    text-align: center;
    border-radius: 10px;
    margin-bottom: 30px;
}

h1 {
    font-size: 2.5rem;
    margin-bottom: 10px;
}

.content {
    background: white;
    padding: 30px;
    border-radius: 10px;
    box-shadow: 0 5px 15px rgba(0,0,0,0.1);
}

.highlight {
    background-color: #ffeaa7;
    padding: 5px 10px;
    border-radius: 5px;
    font-weight: bold;
}

Сохраните этот код в файл styles.css, подключите к вашей HTML-странице и увидите волшебство!
