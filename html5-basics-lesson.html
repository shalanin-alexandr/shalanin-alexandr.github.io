HTML: Строим фундамент и стены

Создаем свою первую веб-страницу и изучаем 5 главных тегов

Уровень

От теории к практике

В прошлом уроке мы разобрались, что HTML — это каркас сайта. Сегодня мы перестанем быть теоретиками. Мы откроем Sublime Text и напишем свою первую страницу.

Ваша цель на этот урок: понять структуру HTML-документа и выучить 5 главных тегов, из которых состоит 90% интернета.

1. Подготовка: Ваш первый файл

Сайт начинается с файла. Давайте создадим его правильно:

Шаг за шагом:

  1. Создайте на рабочем столе папку, назовите её MyFirstSite
  2. Откройте Sublime Text
  3. Нажмите File -> New File (или Ctrl+N)
  4. Сразу сохраните его: File -> Save (или Ctrl+S)
  5. Назовите файл index.html

Почему index? Когда сервер открывает папку с сайтом, он автоматически ищет файл с именем index. Это «главный вход» на ваш сайт.

2. Анатомия тега

HTML состоит из тегов. Тег — это команда для браузера, заключенная в угловые скобки < >.

Большинство тегов — парные. Они работают как контейнер или бутерброд:

<тег>
→
Контент, который внутри
→
</тег>

Объяснение на примере бутерброда

<тег> — Открывающий тег. Это как хлеб снизу. Говорит «начали!».
Контент — Начинка. Текст, картинка, другой тег.
</тег> — Закрывающий тег (со слэшем /). Это как хлеб сверху. Говорит «закончили!».

Важно: Если вы забудете закрыть тег, браузер может запутаться и весь сайт «поедет».

3. Базовая структура (Скелет)

Любой, абсолютно любой HTML-файл должен иметь строгую структуру. Это как паспорт человека — без него никуда.

Вставьте этот код в свой index.html: 

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>Мой первый сайт</title>
    </head>
    <body>

        <!-- Здесь будет контент страницы -->

    </body>
</html>

Разбор скелета

<!DOCTYPE html>

Инструкция для браузера: «Я использую современный HTML5».

<html>

Самый главный контейнер. В нём лежит вообще всё.

<head>

Служебная информация. Тут мы пишем то, что нужно браузеру и поисковикам (кодировка, заголовок вкладки), но пользователь этого не видит на странице.

<body>

А вот здесь лежит всё, что мы видим: текст, картинки, кнопки. Мы будем работать именно внутри <body>.

Лайфхак для Sublime Text: Не нужно писать это вручную каждый раз! Напишите слово html и нажмите клавишу Tab. Редактор сам создаст всю структуру за секунду.

4. Пять главных инструментов строителя

Теперь пишем код внутри тегов <body> и </body>.

№1. Заголовки (Headings)

Они нужны, чтобы выделить тему. Бывают от h1 (самый главный) до h6 (самый маленький).

<h1>Привет, мир! Это главный заголовок</h1>
<h2>Это подзаголовок</h2>
<h3>А это заголовок еще меньше</h3>

Правило: На странице должен быть только один <h1>.

№2. Параграф (Paragraph)

Простой текст мы всегда оборачиваем в тег p. Он создает отступы сверху и снизу, чтобы текст не слипался.

<p>Это обычный текст. Я учусь создавать сайты.</p>
<p>А это уже новый абзац.</p>

№3. Списки (Lists)

В интернете мы обожаем списки (меню, товары, пункты).

  • <ul> (Unordered List) — Маркированный список (с точками)
  • <ol> (Ordered List) — Нумерованный список (1, 2, 3)
  • <li> (List Item) — Сам элемент списка
<ul>
    <li>Хлеб</li>
    <li>Молоко</li>
    <li>Яйца</li>
</ul>

№4. Ссылки (Anchor)

То, что делает интернет «паутиной». Тег <a>. У него есть атрибут href — это адрес, куда мы полетим при клике.

<a href="https://google.com">Погуглить</a>

5. Картинки (Image)

Тег <img>. Обратите внимание: у него нет закрывающего тега! Ему нечего хранить внутри, он просто показывает файл. У него есть атрибут src (source — источник) и alt (alt — это альтернативное текстовое описание содержания изображения, например, если картинка на сайте пользователя не прогрузится).

<img src="https://placekitten.com/200/300" alt="Милый котик">

5. ID и Классы: Именование элементов

Чтобы управлять элементами на странице (стилизовать их, делать интерактивными), нам нужно давать им имена. Для этого в HTML есть два специальных атрибута: id и class.

ID - уникальный идентификатор

id - это как паспорт элемента. Он должен быть уникальным на странице.

<h1 id="main-title">Мой сайт</h1>
<div id="header">...</div>
<img id="profile-photo" src="...">

Особенности ID:

  • Должен быть уникальным (один ID на странице)
  • Используется для якорных ссылок: <a href="#main-title">
  • В CSS обращение через #main-title

Class - группировка элементов

class - это как группа друзей. Один класс можно давать многим элементам.

<p class="text-red">Этот текст красный</p>

<p class="text-blue">Этот текст голубой</p>

<p class="text-red">Этот текст снова красный!</p>

<!-- Значит, мы можем создавать классы с их уникальными стилями, 
а потом передавать эти стили любым элементам на странице! -->

Особенности классов:

  • Можно использовать много раз
  • Элемент может иметь несколько классов (через пробел)
  • В CSS обращение через .text-red
  • Идеально для стилизации групп элементов

Сравнение ID и Class

ID

  • Уникальный (один на странице)
  • Используется для уникальных элементов
  • CSS: #element-id
  • Пример: id="header"

Class

  • Многократный (много на странице)
  • Используется для групп элементов
  • CSS: .element-class
  • Пример: class="button"

Зачем это нужно? В следующем уроке по CSS мы научимся стилизовать элементы по их ID и классам, делая наш сайт красивым и профессиональным!

5. Практика: Собираем всё вместе

Перепишите этот код в свой файл index.html со своей информацией, сохраните (Ctrl+S) и откройте файл в браузере (дважды кликните по файлу в папке). 

<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Моя страница</title>
</head>
<body>

    <header id="main-header">
        <h1 class="site-title">Саша и Веб-разработка</h1>
    </header>

    <main class="content">
    <!-- main показывает основное наполнение страницы - так вы можете обратится ко всему "главному" на странице -->
        <p class="intro-text">Привет! Меня зовут Саша, и я начал изучать HTML.</p>

        <section class="goals-section">
        <!-- section показывает определённую секцию на странице. Например - секция с видеоиграми, секция с музыкой на странице и т.п. -->
            <h2 class="section-title">Мои цели:</h2>
            <ul class="goals-list">
                <li class="goal-item">Понять, как работает веб</li>
                <li class="goal-item">Сделать свой первый сайт</li>
                <li class="goal-item important">Перестать прокрастинировать</li>
            </ul>
        </section>

        <a href="https://youtube.com" class="external-link">Смотреть видео</a>

        <img src="..." alt="Ноутбук и код" class="profile-image">
    </main>

</body>
</html>

Поздравляю! Вы только что создали свою первую веб-страницу!

6. Домашнее задание

Просмотр чужого кода не сделает вас программистом. Нужно писать свой!

Задание: Создать страницу «О себе»

Требования:

В файле должна быть правильная структура (html, head, body)
Использовать заголовок <h1> для имени
Написать пару абзацев текста <p> о биографии или увлечениях
Сделать список <ul> или <ol>
Вставить картинку <img>
Добавить ссылку <a>
Использовать id для главных элементов и class для группировки
