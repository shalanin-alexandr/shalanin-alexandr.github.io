Типы данных в C++

В C++ каждая переменная имеет тип, который определяет, какие данные можно хранить в этой переменной и сколько памяти она занимает.

Основные типы данных:

// Целые числа
int age = 25;              // Целое число (обычно 4 байта)
long bigNumber = 1000000;   // Длинное целое число
short smallNumber = 100;   // Короткое целое число

// Дробные числа
float price = 19.99;        // Число с плавающей точкой (4 байта)
double pi = 3.14159;        // Число с двойной точностью (8 байт)

// Символы и строки
char letter = 'A';           // Один символ
string name = "Иван";        // Строка символов

// Логический тип
bool isStudent = true;      // true или false

Объявление и инициализация переменных

В C++ переменную нужно сначала объявить (сообщить компилятору о её существовании), а затем можно присвоить ей значение. 

// Способ 1: Объявление и инициализация одновременно
int number = 10;

// Способ 2: Сначала объявление, потом присваивание
int number;
number = 10;

// Способ 3: Инициализация при объявлении (современный стиль C++)
int number{10};

Важно:

В C++ переменные не инициализируются автоматически. Если вы объявите переменную без присваивания значения, она будет содержать "мусор" — случайные данные из памяти. Всегда инициализируйте переменные!

Правила именования переменных

  • Имя может содержать буквы, цифры и символ подчеркивания (_)
  • Имя должно начинаться с буквы или подчеркивания
  • Регистр букв имеет значение: age и Age — разные переменные
  • Нельзя использовать зарезервированные слова (int, return, if и т.д.)
  • Имя должно быть понятным и описывать назначение переменной
// Хорошие имена
int studentAge = 20;
string userName = "Иван";
double averageScore = 4.5;

// Плохие имена (непонятные)
int x = 20;
string n = "Иван";
double as = 4.5;

Базовые операции с переменными

С переменными можно выполнять различные математические операции: сложение, вычитание, умножение, деление и другие. 

#include <iostream>
using namespace std;

int main() {
    int a = 10;
    int b = 5;
    
    // Арифметические операции
    int sum = a + b;        // Сложение: 15
    int diff = a - b;        // Вычитание: 5
    int product = a * b;     // Умножение: 50
    int quotient = a / b;    // Деление: 2
    int remainder = a % b;   // Остаток от деления: 0
    
    // Сокращенные операции
    a += 5;  // То же, что a = a + 5
    a -= 3;  // То же, что a = a - 3
    a *= 2;  // То же, что a = a * 2
    a /= 2;  // То же, что a = a / 2
    
    // Инкремент и декремент
    a++;  // Увеличить на 1 (постфиксная форма)
    ++a;  // Увеличить на 1 (префиксная форма)
    a--;  // Уменьшить на 1
    
    return 0;
}

Ввод и вывод данных

Для работы с вводом и выводом данных в C++ используется библиотека iostream. Мы уже использовали cout для вывода. Теперь изучим cin для ввода. 

#include <iostream>
using namespace std;

int main() {
    int age;
    string name;
    
    // Вывод текста
    cout << "Введите ваше имя: ";
    
    // Ввод строки
    cin >> name;
    
    cout << "Введите ваш возраст: ";
    
    // Ввод числа
    cin >> age;
    
    // Вывод результата
    cout << "Привет, " << name << "! Тебе " << age << " лет." << endl;
    
    return 0;
}

Объяснение:

  • cout (console output) — используется для вывода данных на экран
  • cin (console input) — используется для ввода данных с клавиатуры
  • << — оператор вывода (направление данных в поток)
  • >> — оператор ввода (направление данных из потока в переменную)
  • endl — переводит курсор на новую строку

Практический пример

Давайте создадим программу, которая вычисляет площадь прямоугольника: 

#include <iostream>
using namespace std;

int main() {
    double width, height, area;
    
    cout << "Введите ширину прямоугольника: ";
    cin >> width;
    
    cout << "Введите высоту прямоугольника: ";
    cin >> height;
    
    // Вычисление площади
    area = width * height;
    
    cout << "Площадь прямоугольника: " << area << endl;
    
    return 0;
}

Частые ошибки

  • Неинициализированные переменные: Всегда присваивайте начальное значение
  • Неправильный тип данных: Не пытайтесь хранить дробное число в переменной типа int
  • Деление на ноль: Проверяйте делитель перед делением
  • Путаница с операторами: Помните, что = — это присваивание, а не сравнение
  • Забыли точку с запятой: Каждая команда должна заканчиваться ;

Лучшие практики

  • Используйте понятные имена переменных
  • Всегда инициализируйте переменные при объявлении
  • Выбирайте правильный тип данных для вашей задачи
  • Используйте const для переменных, которые не должны изменяться
  • Комментируйте сложные вычисления
